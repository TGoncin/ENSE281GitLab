Git basics (ENSE 281)

Configuration
- git config --global user.name "Your Name"
- git config --global user.email your@email.ca

Initialize repo
- git init -b main
- git status

Stage + commit
- git add <file>
- git add .
- git commit -m "Message"

Remotes
- git remote add origin <URL>
- git remote set-url origin <URL>
- git remote -v

Push / pull
- git push -u origin main
- git push

Branches
- git branch
- git checkout -b alternate
- git checkout main
- git push -u origin alternate

Merge
- git merge alternate
- resolve conflict in file, keep final text
- git add <file>
- git commit -m "Merge alternate so evil prevails"

History / graph
- git log
- git log --oneline
- git log --all --decorate --oneline --graph
